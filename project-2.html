Context-Aware Car Assistant

Built RAG-powered car assistant by indexing a vehicle manual (embeddings + FAISS) to explain dashboard warnings and surface actionable recommendations.

Project Overview

This project involved building a Retrieval-Augmented Generation (RAG) powered car assistant. The assistant indexes a vehicle manual using embeddings and FAISS to explain dashboard warnings and provide actionable recommendations.

I implemented LangChain Expression Language (LCEL) chains, function-calling, and multi-turn memory to ensure the assistant provides contextual and safety-aware responses to user queries.

Tools Used

LangChain
OpenAI API
FAISS
